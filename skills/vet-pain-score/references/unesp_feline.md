# UNESP-Botucatu Multidimensional Composite Pain Scale (Cats)

## Subscale 1: Pain Expression (0-12)
- **Miscellaneous behaviors**: Lying down quiet but moving tail, contracts pelvic limbs/flank, eyes partially closed, licks/bites wound. (None=0 -> All=3)
- **Reaction to palpation (wound)**: No reaction(0) -> Does not allow palpation(3).
- **Reaction to palpation (abdomen)**: No reaction(0) -> Does not allow palpation(3).
- **Vocalization**: Quiet/purring(0) -> Growls/howls spontaneously(3).

## Subscale 2: Psychomotor Change (0-12)
- **Posture**: Natural/relaxed(0) -> Frequently alters position(3).
- **Comfort**: Comfortable(0) -> Uncomfortable/restless/facing back of cage(3).
- **Activity**: Moves normally(0) -> Reluctant to move(3).
- **Attitude**: Satisfied/Alert(0) -> Aggressive/tries to bite(3).

## Subscale 3: Physiological (0-6)
- **Arterial BP**: 0-15% above pre-surgery(0) -> >45% above(3).
- **Appetite**: Eating normally(0) -> Not interested in food(3).

## Thresholds
- **Analgesia recommended**: Total Score >= 6 / 30.
- **Rescue dose needed**: High scores in Subscale 1.
