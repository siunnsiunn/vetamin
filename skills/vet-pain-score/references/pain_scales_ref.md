# Veterinary Pain Scales Reference (PAIN_SCALES.md)

## 1. Feline Grimace Scale (FGS)
適用於貓咪急性疼痛。評估 5 個 Action Units (AU)，每項 0-2 分：
- **Ear position**: 0 (Ears forward) -> 2 (Ears pulled apart/flattened)
- **Orbital tightening**: 0 (Eyes open) -> 2 (Squinted/closed)
- **Muzzle tension**: 0 (Relaxed/round) -> 2 (Tense/elliptical)
- **Whisker position**: 0 (Loose/curved) -> 2 (Straight/pointing forward)
- **Head position**: 0 (Above shoulder) -> 2 (Lowered/tilted)
**Intervention Threshold**: Score >= 0.4 / 1.0 (或總分 >= 4/10) 建議投藥。

## 2. Glasgow Composite Measure Pain Scale (CMPS)
### Feline (Short Form)
包含：Vocalisation, Posture, Activity, Attention to wound, Ear position, Muzzle shape, Response to interaction, Palpation.
**Intervention Threshold**: Score >= 5 / 20 建議投藥。

### Canine (Short Form)
包含：Vocalisation, Attention to wound, Mobility, Response to palpation, Body posture, Activity.
**Intervention Threshold**: Score >= 6 / 24 建議投藥。

## 3. Chronic Pain (DJD/OA)
- **FMPI (Feline Musculoskeletal Pain Index)**: 追蹤 Jumping, Stairs, Play, Activity.
- **LOAD (Liverpool Osteoarthritis in Dogs)**: 評估穩定性與運動後的 Lameness。
