# Feline Musculoskeletal Pain Index (FMPI) - Full Questionnaire

## Scoring Instructions
- **Questions 1-17**: Score each from 4 (Normal/Best) to 0 (Severe/Worst).
- **Questions 18-21**: Qualitative "Yes/No" indicators for clinical assessment.
