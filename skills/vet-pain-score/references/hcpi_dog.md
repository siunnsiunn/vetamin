# Helsinki Chronic Pain Index (HCPI) - Owner Questionnaire

1. Attitude/Mood: Very alert(0) -> Very lethargic(4)
2. Play/Interact: Very willing(0) -> Does not participate(4)
3. Vocalization: Never(0) -> Very often(4)
4. Eagerness to walk: Very eager(0) -> Does not want to(4)
5. Stairs ability: Very willing/able(0) -> Does not do stairs(4)
6. Run ability: Very willing/able(0) -> Does not run(4)
7. Jump ability: Very willing/able(0) -> Does not jump(4)
8. Ease in lying down: Very easy(0) -> Very difficult(4)
9. Rising from down: Very easy(0) -> Very difficult(4)
10. Movement after rest: Very easy(0) -> Very difficult(4)
11. Movement during/after exercise: Very easy(0) -> Very difficult(4)
