# 獸醫毒物去污染與急救快速參考 (Toxicology Quick Ref)

本指南摘錄自《Blackwell's Five-Minute Veterinary Consult Clinical Companion: Small Animal Toxicology, 3rd Edition》。

## 1. 現場與診間去污染 (Decontamination)
*   **眼部 (Ophthalmic)**：立即使用洗眼液或生理食鹽水沖洗 15-20 分鐘。
*   **呼吸道 (Respiratory)**：移至新鮮空氣處，必要時給予純氧或人工通氣。
*   **皮膚 (Dermal)**：使用溫和洗碗精（如 Dawn）大量沖洗，避免毒素經皮吸收。
*   **消化道 (Gastrointestinal)**：
    *   **催吐 (Emesis)**：僅限於誤食 2-4 小時內且病患意識清醒、具備正常吞嚥反射。
    *   **活性碳 (Activated Charcoal)**：1-4 g/kg PO。注意：醇類、強酸、強鹼、重金屬效果不佳。

## 2. 常見毒物臨床特徵 (Toxidromes)
*   **木糖醇 (Xylitol)**：
    *   **特徵**：急性低血糖（誤食後 30-60 分鐘內）、延遲性肝衰竭。
    *   **處置**：立即監測血糖，必要時補充 Dextrose。
*   **乙二醇 (Ethylene Glycol)**：
    *   **特徵**：代謝性酸中毒、高滲透壓間距 (Osmolar Gap)、急性腎小管壞死 (ATN)。
*   **巧克力/咖啡因 (Methylxanthines)**：
    *   **特徵**：心搏過速、顫抖、過度興奮。
*   **百合花 (Lilies - 貓隻專屬)**：
    *   **特徵**：急性腎衰竭。即使只吃一片花瓣也具致命性。

## 3. 重要解毒劑與藥物劑量 (Antidotes)
| 毒物類別 | 解毒劑 / 藥物 | 劑量與備註 |
| :--- | :--- | :--- |
| **Alpha-2 激動劑** | Atipamezole | 建議 IM 給藥，避免 IV 引起的心動過速。 |
| **有機磷/胺基甲酸鹽** | Atropine | 0.1-0.5 mg/kg (1/4 IV, 其餘 IM/SQ)。僅針對 SLUDGE 症狀。 |
| **苯二氮平類 (Benzos)** | Flumazenil | 拮抗劑。 |
| **鐵 (Iron)** | Deferoxamine | 螯合治療。 |
| **維生素 D3** | Pamidronate | 針對高血鈣治療。 |
| **氰化物 (Cyanide)** | Hydroxocobalamin | 首選解毒劑。 |

## 4. 治療查核點
*   **Fluid Diuresis**：多數中毒病患需積極靜脈輸液以促進毒素排出（尤其是維生素 C 或藥物過量）。
*   **Seizure Control**：必須積極控制抽搐，以防腦水腫與吸入性肺炎。
*   **Thermoregulation**：注意控制震顫引起的體溫過高或代謝受損引起的體溫過低。
