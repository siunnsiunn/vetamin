# FECAVA / ALIVE 2024 Diabetes Management Protocols

## 1. 診斷標準 (ALIVE Criteria)
### 犬 (Canine)
- 隨機 BG >= 200 mg/dL + 典型症狀 (PU/PD, Weight loss).
### 貓 (Feline)
- 隨機 BG >= 270 mg/dL + 典型症狀 + (糖化蛋白升高 OR 兩次居家尿糖陽性).
- 或是 BG 130-270 mg/dL 之間需合併兩項以上診斷標準。

## 2. 胰島素起始劑量 (Starting Doses)
| Insulin Type | Dog (IU/kg) | Cat (IU/cat or IU/kg) | Frequency |
| :--- | :--- | :--- | :--- |
| Glargine 100 (Lantus) | 0.3 IU/kg | 1.0 - 1.5 IU/cat | q12h |
| Glargine 300 (Toujeo) | 0.5 IU/kg | 0.5 IU/kg | q12h (q24h) |
| Lente (Caninsulin) | 0.25 IU/kg | 1.0 - 1.5 IU/cat | q12h (q8-12h) |

## 3. 調整邏輯 (Adjustment Logic)
- **良好的控制 (Good Control)**: Nadir 80-150 mg/dL (4.4-8.3 mmol/L).
- **調整幅度**:
    - 貓: 0.5 - 1.0 IU / injection.
    - 狗: 10% - 25% / injection.
- **臨床低血糖 (< 60 mg/dL)**: 
    - 停藥直到 BG > 180 mg/dL.
    - 以 **50% 劑量** 重新啟動治療。
