# BSAVA Liver Interpretation (LIVER.md)

## 1. Enzyme Patterns
- **Hepatocellular Leakage (ALT, AST)**:
    - 代表肝細胞受損（發炎、毒素、缺氧）。
    - **ALT**: 肝臟特異性最高（犬貓）。
- **Cholestasis (ALKP, GGT)**:
    - 代表膽汁淤積。
    - **Feline Note**: 貓咪 ALKP 半衰期極短 (6h)，即使輕微上升也具有重大意義。
    - **ALKP ↑ + GGT normal**: 常見於貓咪 Hepatic Lipidosis (脂肪肝)。

## 2. Liver Function Markers (肝功能指標)
當以下 4 項同時下降時，代表肝臟功能剩不到 30%：
- **ALB ↓** (白蛋白)
- **BUN ↓** (血氨微縮)
- **CHOL ↓** (膽固醇)
- **GLU ↓** (血糖)

## 3. Advanced Tests
- **Bile Acids (BA)**: 評估肝門脈分流 (PSS) 或肝硬化。
- **Ammonia (血氨)**: 評估 Hepatic Encephalopathy (肝腦症)。
- **Bilirubin (膽紅素)**: 區分 Pre-hepatic (Hemolysis), Hepatic, Post-hepatic (Obstruction)。
