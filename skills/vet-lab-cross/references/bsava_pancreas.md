# BSAVA Pancreas Interpretation (PANCREAS.md)

## 1. Exocrine Pancreatitis (胰臟炎)
- **Traditional Markers**: Amylase / Lipase (特異性較低)。
- **Precision Markers**: **Spec cPL** (Canine) / **Spec fPL** (Feline)。
    - 臨床金標準：Spec fPL > 5.4 µg/L 強烈建議胰臟炎。
- **Secondary Changes**: 
    - Stress Leukogram.
    - Hypocalcaemia (常見於貓咪嚴重胰臟炎)。
    - Hyperglycaemia (暫時性)。

## 2. EPI (Exocrine Pancreatic Insufficiency)
- **Primary Indicator**: **TLI (Trypsin-like Immunoreactivity)**。
    - **Low TLI**: 確診 EPI。
- **Associated Findings**: 
    - **Cobalamin (B12) ↓**: 因缺乏內在因子吸收。
    - **Folate (葉酸) ↑**: 常伴隨 SIBO (小腸細菌過度增生)。
