# BSAVA Urinary System Interpretation (URINARY.md)

## 1. Azotaemia Differentiation (氮質血症區分)
- **Pre-renal**: CREA ↑ + USG 高 (>1.030 Dog / >1.035 Cat)。主因是脫水。
- **Renal**: CREA ↑ + USG 低 (<1.030 Dog / <1.035 Cat)。主因是腎實質受損。
- **Post-renal**: 臨床症狀 (無尿/排尿困難) + 影像學 (結石/阻塞)。

## 2. IRIS CKD Staging (進階)
除了 CREA，應結合 **SDMA** 進行早期評估：
- **Stage 1**: CREA < 140 (Cat) / < 125 (Dog) 但 SDMA > 14。
- **Note**: 評估分級前必須確保 **Patient is hydrated and stable**。

## 3. Proteinuria (蛋白尿)
主要依據 **UPC (Urine Protein:Creatinine Ratio)**：
- **UPC > 0.4 (Cat) / > 0.5 (Dog)**: 確診為顯著蛋白尿。
- **Differentials**: 排除發炎 (Sediment active) 後，考慮 PLN (Glomerular disease)。

## 4. Urinalysis Key Indicators
- **Glucosuria with Normoglycaemia**: 考慮 Fanconi syndrome 或近端腎小管受損。
- **Casts (管型)**: 代表腎小管急性損傷 (Active tubular damage)。
