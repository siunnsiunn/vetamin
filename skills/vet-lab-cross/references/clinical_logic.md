# Veterinary Clinical Reasoning Logic (LCPS_LOGIC.md)

## 1. Defining the Diagnostic Hook (硬指標)
優先處理特異性最高的異常，而不是最嚴重的臨床症狀。
- **Primary Structural Issue**: 需要影像學 (Imaging) 或採樣 (Biopsy)。
- **Secondary Functional Issue**: 需要血液生化 (Lab Work) 或內分泌測試。

## 2. Refine the System (系統定位)
- **Urinary**: Azotemia + USG low.
- **Hepatobiliary**: ALT/ALKP high + Icterus.
- **Gastrointestinal**: Vomiting/Diarrhea + Electrolyte shift.
- **Hematologic**: Anemia/Leukogram changes.

## 3. DAMNIT-V Differential Framework
當系統確定後，按以下順序排除：
- **D**egenerative (退化)
- **A**nomalous (先天)
- **M**etabolic (代謝)
- **N**eoplastic (腫瘤) / **N**utritional (營養)
- **I**nflammatory (發炎) / **I**nfectious (感染) / **I**mmune-mediated (免疫) / **I**diopathic (特發) / **I**atrogenic (醫源)
- **T**raumatic (創傷) / **T**oxin (毒素)
- **V**ascular (血管)
