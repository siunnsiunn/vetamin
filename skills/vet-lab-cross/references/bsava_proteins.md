# Plasma Proteins Interpretation

## 1. Key Markers

### Total Protein (TP)
- **Composition**: Albumin (ALB) + Globulins (GLOB)。
- **Measurement**:
  - Biochemistry analyzer: Colorimetric reaction，較為精確。
  - Refractometer: 利用 refractive index 測量。**Lipaemia** 與 **Haemolysis** 會嚴重干擾 refractometer，導致數值 false marked elevation。

### Albumin (ALB)
- **Characteristics**: 由肝臟製造，佔 plasma oncotic pressure 的 75-80%。也是許多 drugs 與 calcium 的重要 carrier protein。
- **Half-life**: 狗約 8 天。
- **Hyperalbuminaemia**: 
  - **唯一的 clinical cause 就是 Dehydration**。因為肝臟不可能「過度製造」albumin。
- **Hypoalbuminaemia**: 
  - **Decreased production**: Hepatic failure (>75% function lost, 如 Cirrhosis, PSS)、Chronic inflammation (Acute phase response 中，肝臟會減少製造 ALB 以轉向製造 positive acute phase proteins)。
  - **Increased loss**: 
    - Renal loss (Protein-Losing Nephropathy, PLN)
    - GI loss (Protein-Losing Enteropathy, PLE)
    - 大面積 skin burns、severe exudative effusion (如 peritonitis)。
  - **Haemodilution**: IV fluid overload。

### Globulins (GLOB)
- **Composition**: 包括 Alpha, Beta, Gamma (Immunoglobulins) 三大類。通常由 TP 減去 ALB 計算得出。
- **Hypoglobulinaemia**: 
  - 常見於 neonates (Failure of passive transfer)、severe haemorrhage、PLE (通常合併 ALB ↓)。
- **Hyperglobulinaemia**:
  - 代表 acute/chronic inflammation，或 immune system over-activation (如 infection, immune-mediated disease, neoplasia)。

### A:G Ratio (Albumin/Globulin Ratio)
- **Feline Note**: A:G Ratio < 0.4 - 0.6 且 GLOB 高時，需強烈懷疑 **FIP (貓傳染性腹膜炎)**。

## 2. Clinical Patterns

### 2.1 Panhypoproteinaemia
- **Characteristics**: ALB 與 GLOB 同時 decreased。
- **Differentials**:
  - **Protein-Losing Enteropathy (PLE)**: 腸道黏膜 non-selective leakage。
  - **Severe Haemorrhage**: 血液流失後，interstitial fluid 回流導致 haemodilution。
  - **Severe Exudative Effusion**: 如 FIP、severe peritonitis，proteins 大量跑到 third space。

### 2.2 Hyperglobulinaemia
Hyperglobulinaemia 需透過 **Serum Protein Electrophoresis (SPE)** 來區分是 polyclonal 還是 monoclonal：

1. **Polyclonal Gammopathy**:
   - **Electrophoresis Pattern**: Gamma 區段呈現寬闊、圓滑的波峰 (Broad-based peak)。代表多種 B cells 同時被活化。
   - **Differentials**: 
     - **Chronic infection**: FIP、Dirofilariasis、Ehrlichiosis、Leishmaniosis。
     - **Immune-mediated disease**: IMHA、IMPA。
     - **Chronic inflammation**: Severe periodontal disease、deep abscesses。
2. **Monoclonal Gammopathy**:
   - **Electrophoresis Pattern**: Gamma (或 Beta) 區段出現極窄、尖銳的波峰 (Narrow, tall peak, "Church spire")。代表單一個 B cell 或 Plasma cell 發生 neoplastic proliferation。
   - **Differentials**: 
     - **Multiple Myeloma**: 最常見。
     - B-cell Lymphoma。
     - 罕見感染 (如某些 Ehrlichiosis 可能表現為 monoclonal)。

### 2.3 Acute Phase Proteins (APPs)
- 肝臟在 inflammatory response 時會改變蛋白質製造的優先順序。
- **Positive APPs (發炎時 increased)**: 
  - C-Reactive Protein (CRP, 狗的 sensitive marker)。
  - Serum Amyloid A (SAA, 狗貓皆有)。
  - Fibrinogen。
  - Haptoglobin。
- **Negative APPs (發炎時 decreased)**: 
  - Albumin。
  - Transferrin。

## 3. Important Artifacts & Considerations
- **TP False Elevation**: Lipaemia 與 Haemolysis 對 Refractometer 的 interference 極大。如果抽血時發現血清紅紅或濁濁的，測出來的高 TP 是 unreliable 的。
- **Age Differences**: 幼年動物的 TP 與 GLOB 本來就較低，而 ALKP 與 Phosphorus 本來就較高 (因 bone growth)，判讀時不可套用 adult reference intervals。
