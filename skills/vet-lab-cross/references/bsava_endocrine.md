# Endocrine Disease Interpretation

## 1. Key Markers

### Glucose (GLU)
- **Hyperglycaemia**:
  - **Stress/Excitement**: 貓極為常見，因 Epinephrine release，血糖可瞬間飆高至 300 mg/dL 以上。狗的 stress hyperglycaemia 通常較輕微。
  - **Diabetes Mellitus (DM)**: Persistent hyperglycaemia 伴隨 Glucosuria。
  - **Diestrus/Pregnancy**: 狗因 Progesterone 拮抗 insulin。
  - **Hyperadrenocorticism (Cushing's)**: Glucocorticoids 拮抗 insulin。
- **Hypoglycaemia**:
  - **Increased utilization**: Sepsis、Hunting dog hypoglycemia (劇烈運動消耗)。
  - **Decreased production**: Hepatic failure (PSS, Cirrhosis)、Toy breed/Neonatal hypoglycemia (幼犬/小型犬 hepatic glycogen storage 不足)、Hypoadrenocorticism (Addison's disease)。
  - **Excess insulin**: Insulinoma、Xylitol toxicity (木糖醇中毒，狗)、Insulin overdose。

### Thyroid Hormones
- **Total T4 (TT4)**: 評估 thyroid function 的 first-line marker。
- **Free T4 (fT4)**: 游離態的 T4，不受 protein binding 影響。Equilibrium dialysis (透析法) 是最準確的測量方式。
- **cTSH (Canine Thyroid Stimulating Hormone)**: 輔助診斷 hypothyroidism 的 marker。

### Adrenal Hormones
- **Cortisol**: 需透過 Dynamic testing 評估，單次抽血測量 basal Cortisol 的 diagnostic value 極低 (僅能用來排除 Addison's，若 > 2.0 µg/dL 則極不可能為 Addison's)。

## 2. Clinical Patterns

### 2.1 Diabetes Mellitus (DM) vs. Stress Hyperglycaemia
- 單次的 hyperglycaemia 與 glucosuria 在貓無法區分 DM 與 stress。
- **Fructosamine**: 測量過去 2-3 週內的 average blood glucose concentration。
  - DM：Fructosamine 顯著 increased。
  - Stress：Fructosamine normal。
  - **Caution**: Hypoproteinaemia 或 Hyperthyroidism (蛋白質代謝加速) 會導致 Fructosamine 假性偏低，可能掩蓋早期的 DM。

### 2.2 Canine Hypothyroidism
- **Laboratory Pattern**:
  - 常併發 **Hypercholesterolaemia** 與 **Hypertriglyceridaemia**。
  - 輕度 Non-regenerative Anaemia。
- **Diagnosis**: 
  - **Low TT4 + High cTSH**: 最支持 primary hypothyroidism。
  - **Euthyroid Sick Syndrome**: 嚴重的 systemic disease (或使用 corticosteroids, Phenobarbital) 會 suppress thyroid axis，導致 TT4 低下，但 cTSH 通常 normal。若懷疑此症，應加驗 fT4 (通常 normal)。

### 2.3 Feline Hyperthyroidism
- **Laboratory Pattern**:
  - **ALT / ALKP / AST increased**: 幾乎所有 hyperthyroid cats 都有 liver enzymes 輕至中度上升。
  - 經常併發 Stress leucogram 或 Erythrocytosis。
- **Diagnosis**:
  - **High TT4**: 確診 hyperthyroidism。
  - 若 TT4 落在 normal upper limit (Grey zone) 但臨床高度懷疑，可加驗 fT4 (較敏感) 或進行 T3 suppression test。

### 2.4 Hyperadrenocorticism (HAC / Cushing's Disease)
- **Laboratory Pattern**:
  - **ALKP 顯著 increased**: 狗的 specific presentation，因 corticosteroids 誘導 isoenzyme (C-ALP) 大量製造。
  - Stress Leucogram (SMILED)。
  - **USG < 1.015**: Corticosteroids 拮抗 ADH 導致 polyuria，經常呈現 Hyposthenuria。
- **Diagnosis**:
  - **LDDST (Low-Dose Dexamethasone Suppression Test)**: Sensitivity 高，為 Screening 的 priority choice。**Pattern**: 8h cortisol > 1.4 µg/dL (Fail to suppress)。
  - **ACTH Stimulation Test**: Specificity 高，常用於 diagnostic confirmation 或 monitoring (如 Trilostane therapy)。

### 2.5 Hypoadrenocorticism (Addison's Disease)
- **Laboratory Pattern**:
  - **Na:K ratio < 27 (常 < 24)**: Hyponatraemia + Hyperkalaemia。
  - Lack of stress leucogram (即使動物病得很重，lymphocytes 與 eosinophils 卻 normal 或偏高)。
  - 常併發 Pre-renal / Renal Azotaemia、Hypoglycaemia、Hypercalcaemia。
- **Diagnosis**:
  - **ACTH Stimulation Test**: 唯一 gold standard。**Pattern**: Flat response (Post-ACTH cortisol < 2 µg/dL)。

## 3. Important Artifacts & Considerations
- **Sample Handling (Glucose)**: Whole blood 樣本若未及時分離 serum/plasma，紅血球會持續消耗 glucose (約 10% / hour)。必須盡快 centrifugation 或使用 Fluoride 採血管。
- **Sighthounds & T4**: 視覺獵犬 (如 Greyhounds, 灰狗) 的 normal TT4 reference interval 較其他犬種低很多，不可誤判為 Hypothyroidism。
- **Euthyroid Sick Syndrome**: 不要單憑一次生病時的 low TT4 就診斷為 hypothyroidism 並開始 thyroid supplementation，這會加重動物的 metabolic burden。
